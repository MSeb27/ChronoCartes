## Idées d'extensions (paquets thématiques)

- **Paquet France** : rois, batailles, républiques, lois clés.
- **Paquet Sciences & inventions** : pour resserrer les écarts et corser le jeu.
- **Paquet Sport / JO**, **Paquet Espace**, **Paquet Arts & culture**.
- **Cartes locales** : événements choisis par les joueurs (anniversaire, fondation d'un club…) pour personnaliser le paquet.
